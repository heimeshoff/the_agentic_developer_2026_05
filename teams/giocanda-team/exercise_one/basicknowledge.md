# Personal Finance App - Basic Knowledge

## Extracted Features from Workshop Material

This document captures the core features identified for the personal finance and budgeting application.

### Core Features

1. **Income Tracking**
   - Monitor money coming in from various sources
   - Support multiple income streams
   - Track regular vs irregular income

2. **Budgeting**
   - Plan and allocate funds across different categories
   - Set monthly/weekly budget limits
   - Compare planned vs actual spending

3. **Expense Tracking**
   - Record and categorize spending
   - Track where money is going
   - Support various expense categories

4. **Savings Goals**
   - Set and track progress toward savings targets
   - Define specific savings objectives
   - Monitor progress toward goals

5. **Investment Tracking**
   - Monitor investment portfolios and performance
   - Track returns and growth
   - Visualize investment allocation

6. **Bill Management**
   - Track recurring bills and payment due dates
   - Set up reminders for upcoming payments
   - Monitor payment history

7. **Financial Reports**
   - Generate insights and analytics on financial health
   - Provide overview of financial status
   - Identify spending patterns and trends

8. **Notifications/Alerts**
   - Remind users about bills
   - Alert when approaching budget limits
   - Notify about savings milestones

9. **Account Management**
   - Manage multiple bank accounts
   - Track credit cards
   - View balances across accounts

10. **Transaction History**
    - View and search past transactions
    - Filter by date, category, amount
    - Transaction details and notes

11. **Categories & Tags**
    - Organize expenses and income with custom categories
    - Tag transactions for better organization
    - Create category hierarchies

12. **Financial Goals**
    - Set and track broader financial objectives
    - Beyond just savings (e.g., debt reduction, net worth targets)
    - Long-term financial planning

### Supporting Features

- **Dashboard/Overview**
  - Central view of overall financial status
  - Key metrics at a glance
  - Quick access to important information

- **Data Visualization**
  - Charts and graphs for spending patterns
  - Income vs expenses visualization
  - Trend analysis over time

- **Multi-currency Support** (optional)
  - Support for different currencies
  - Currency conversion
  - International transactions

- **Export/Import**
  - Data portability
  - Backup and restore
  - Integration with other tools
